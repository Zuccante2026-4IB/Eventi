# Eventi - Gestione degli eventi

# 📌 Progetto Eventi

Applicazione per la gestione, visualizzazione e filtraggio di eventi, con funzionalità avanzate di ricerca, ordinamento e interazione utente.

---

## 🚀 Funzionalità principali

### 🔧 Metodi essenziali

* **Richiesta API**

  * Utilizzo di `fetch()` con `async/await`
  * Responsabili: *Alvise*, *Zai*

* **Gestione lettura/scrittura dati**

  * Responsabile: *Alvise*

* **Scroll a caricamento (infinite scroll)**

  * Responsabili: *Zai*, *Willy*

---

## 🔍 Filtri e Ordinamento

L'utente può filtrare e ordinare gli eventi secondo diversi criteri:

* **📍 Luogo**

  * Responsabile: *Giuseppe*

* **📅 Data / Orario**

  * Personalizzabile dall’utente
  * Responsabile: *Rossi*

* **🎭 Tipo di evento**

  * Responsabile: *Giuseppe*

* **🏢 Gestore evento**

  * Responsabile: *Rossi*

* **💰 Prezzo**

  * Responsabile: *Giuseppe*

* **🎯 Target**

  * Responsabile: *Rossi*
